## 🗒 About

This is an app from which you can Explore Any category image and share it on any social media with dark & light Theme.

## ⚙️ Features

- Main features are as follows:

    - Splash Page
    - Image Load
    - Share Image Link Functionality
    - CategoryVise Image
    - Search Image Functionality
